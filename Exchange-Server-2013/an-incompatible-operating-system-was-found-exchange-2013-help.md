# 找到不相容的作業系統

 

_**適用版本：**Exchange Server_

_**上次修改主題的時間：**2014-01-30_

Microsoft Exchange Server 2013 安裝程式無法繼續，因為偵測到不相容的作業系統。在安裝 Exchange 2013 之前，您必須先在此電腦上安裝相容的作業系統。下表顯示與 Exchange 2013 相容的作業系統。

</thead>

<tr class="odd">
